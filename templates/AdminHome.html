Admin Interface

Welcome to the Admin Interface for this site

This interface allows you to alter the data within the database of this system allowing you to:

  • Remove the log of a Purchase
  • Remove the User data
  • Delete the data within a specific table
  • Delete the data within all tables
  • Refill the tables with fixed, specified data

The different pages allowing these changes can be found using the Navigation Bar at the top of this page
